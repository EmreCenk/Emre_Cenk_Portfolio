Instagram Automation Bot

An Instagram Bot designed to automate tasks and analyze account statistics. This program is equipped with many functions, such that can it can do virtually any task that can be done by a user in Instagram. The project came to life when I could not find any service that would offer account analytics without giving them my password. I also created a website where anyone can use the bot for free, and without giving any passwords.

Dank Miner

I wrote a bot that mines dank memer coins. Dank memer is a currency game that people play via a discord bot. At the peak of dank memer's popularity, I created this project to automate currency mining and betting in dank memer. I wrote minimax algorithms for the gambling games that dank memer offers. As a result, the bot always plays the most optimal move no matter the situation. The approximate mining capacity is around 100,000 coins / hour. What would previously take days, sometimes even weeks can now be done in a few hours thanks to this bot.
